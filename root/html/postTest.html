POST test

Our server does not support POST requests sent via multipart form-data!

However, you can still test it by sending a file through the terminal using the following command:

curl -X POST --data-binary @path/to/file localhost:1234/path/to/endpoint

You can try to send it in chunks (in case you defined in the configs max body size to be smaller than the file you want to send) by adding the following flag to the above command:

-H "Transfer-Encoding: chunked"
