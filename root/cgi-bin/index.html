Let's dive into CGI!

CGI (Common Gateway Interface) is a protocol that allows web servers to execute scripts to process dynamic requests. It is used to generate dynamic content and interact with users through HTTP-based operations.

For POST request, you test it by sending a file through the terminal using the following command:

curl -X POST --data-binary @path/to/file localhost:1234/cgi-bin/post.py

You can try to send it in chunks (in case you defined in the configs max body size to be smaller than the file you want to send) by adding the following flag to the above command:

-H "Transfer-Encoding: chunked"
